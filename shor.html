Approach 2 — How Shor's Actually Works
Finding the Hidden Period
Shor's reduces factoring to period-finding. For a random base a, the function ax mod N repeats with some period r. Classical computers compute each value one at a time. The quantum computer evaluates all values in superposition, then uses a Quantum Fourier Transform to extract the period from the interference pattern — like tuning a radio to find a hidden frequency.

Approach 3 — The Consequence
Why This Changes Everything
As key size grows, classical factoring time explodes exponentially while Shor's grows as a gentle polynomial. At RSA-2048 (the current standard), classical takes longer than the age of the universe. Shor's would take hours.
Classical (best known) Shor's algorithm

Shor's algorithm needs thousands of error-corrected logical qubits. Current quantum computers have ~100–1000 noisy physical qubits — far from enough. But the threat is real enough that NIST has already standardized post-quantum cryptographic algorithms.
